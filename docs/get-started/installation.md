
# Installation

Instructions for installing `kernpy` are provided below.

## Install using `pip`

Update `pip` to the latest version:
```bash
pip install --upgrade pip
```

Install `kernpy` using `pip`:

```bash
pip install kernpy
```

Check your `kernpy` version:

```bash
python -m kernpy --version 
```
